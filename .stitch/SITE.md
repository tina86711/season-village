# Site Vision: 元利四季莊園 (Season Village)

## 1. Vision
A premium landing page for Taipei's largest luxury residential project. The site serves as a digital brochure that communicates architectural elegance, ecological harmony, and top-tier amenities.

## 2. Technical Stack
- **Frontend:** Vanilla HTML5, Vanilla CSS3, Vanilla JavaScript.
- **Icons:** Lucide Icons.
- **Fonts:** Google Fonts (Noto Serif TC, Noto Sans TC, Cormorant Garamond).
- **Standards:** Following `stitch-skills` for design and structure.

## 3. Sitemap
- [x] **Index** (`index.html`) — Main landing page with Hero, Concept, Location, Amenities, Floor Plans, and Contact.

## 4. Roadmap
- [ ] **Mobile Optimization** — Refine tap targets and mobile navigation.
- [ ] **Performance Audit** — Optimize image loading (LCP) and scroll performance.
- [ ] **SEO Enhancement** — Improve metadata and semantic structure.
- [ ] **Dynamic Data Integration** — (Future) Fetch real-time availability.

## 5. Creative Freedom / Ideas
- **Parallax Layers**: Subtle depth in the "Forest" section.
- **Interactive Floor Plan**: Modal views for specific room details.
- **Video Background**: Drone footage integration for the Hero section.
